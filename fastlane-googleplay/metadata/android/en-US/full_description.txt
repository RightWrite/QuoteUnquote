Quote Unquote is an App Widget that displays Quotations.

Discover over twenty thousand, curated, &#34;high quality&#34; quotations.

Key features:
&bull; Filter quotations
&bull; Import quotations
&bull; Customise appearance
&bull; Toolbar navigation
&bull; Notifications
&bull; Backup &amp; Restore
&bull; Wikipedia integration

Other items of interest:
&bull; Free &amp; without Ads
&bull; Core functionality works offline
&bull; Minimal permissions
&bull; Anonymous analytics
&bull; Internal Database content is English
&bull; Open-source software: use GitHub to report defects / raise feature requests - https://github.com/jameshnsears/QuoteUnquote/issues
