<b>Description</b>
Quote Unquote is an App Widget that displays &#34;Quotations&#34;.

The App contains 20,000+ &#34;high quality&#34; quotations.

<b>Key features</b>
&bull; Filter quotations
&bull; Import quotations
&bull; Change appearance
&bull; Toolbar navigation
&bull; Schedule Notifications
&bull; Backup &amp; Restore
&bull; Wikipedia integration

<b>Other items of interest</b>
&bull; Core functionality works offline
&bull; Minimal permissions
&bull; Internal database content is English
&bull; Use GitHub to report defects / raise feature requests
